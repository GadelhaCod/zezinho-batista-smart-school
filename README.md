# Zezinho Batista Smart School

## 📌 Descrição do Projeto
O **Zezinho Batista Smart School** é um sistema web desenvolvido com o objetivo de auxiliar na organização e gestão de informações no ambiente escolar. A aplicação permite o cadastro e gerenciamento de dados relacionados a alunos, professores e turmas, facilitando o acesso e a manipulação dessas informações por meio de uma interface simples e funcional.

O projeto foi desenvolvido como uma iniciativa acadêmica voltada para o aprendizado de desenvolvimento web e aplicação prática de conceitos de programação, estrutura de páginas e manipulação de dados.

---

## 🎯 Objetivo do Projeto
O objetivo do projeto é desenvolver uma aplicação que simule funcionalidades básicas de um **sistema de gestão escolar**, permitindo organizar e visualizar informações importantes de maneira digital.

Entre os principais objetivos estão:

- Aplicar conceitos de desenvolvimento web;
- Criar uma interface simples para gerenciamento de dados escolares;
- Simular funcionalidades de um sistema de administração escolar;
- Praticar organização de código e estrutura de projetos;
- Utilizar o GitHub para versionamento e compartilhamento do projeto.

---

## ⚙️ Funcionalidades
O sistema possui funcionalidades voltadas para a organização de dados escolares, como:

- Cadastro de alunos  
- Cadastro de professores  
- Organização de turmas  
- Visualização de informações cadastradas  
- Interface web para interação com os dados
